# ParseAcademy

ParseAcademy is a lightweight World of Warcraft add-on that adds a right-click menu option for player units and opens a ready-to-copy ParseAcademy character link.

## Usage

- Right-click a player, friend, party member, or raid member.
- Select **Copy Parse Academy Link**.
- Press `Ctrl+C` in the link box, then `Escape` to close it.

Slash commands are also available:

- `/pa` or `/parseacademy` creates a link for your current target.
- `/pa player` or `/pa me` creates a link for your character.
- `/pa focus` creates a link for your focus target.

## Install

Place this folder at:

```text
World of Warcraft/_retail_/Interface/AddOns/ParseAcademy
```

Then enable **ParseAcademy** from the in-game AddOns list.
